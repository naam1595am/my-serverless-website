Vending Machine

I made this vending machine simulation in my CSCE 314 (Progamming Languages) class. For this project I used Java's Queue Interface to store various objects that represent possible items in the vending machine. I also implemented the logic that would delay the complete depletion of a single slot in order to preserve the appearance of a fully stocked vending machine. For the UI design, I used JavaFX and SceneBuilder.
